### 139. Word Break
#### 检测一个字符串是否由字典中的词组成

思路一：动态规划，声明一个数组，存储dp[i]为s[:i]是否由字典词组成

每次判断所有的由字典词组成的字符串加上后面的字符串是否在字典内

```python
class Solution(object):   
    def wordBreak(self, s, wordDict):
        """
        :type s: str
        :type wordDict: List[str]
        :rtype: bool
        """
        dp = [False for _ in range(len(s)+1)]
        dp[0] = True
        
        for i in range(1, len(s)+1):
            if s[:i] in wordDict:
                dp[i] = True
                continue
            for j in range(1, i):
                if dp[j]:
                    if s[j:i] in wordDict:
                        dp[i] = True
                        break
        return dp[-1]
```
每次判断从最大的字典词的长度往前遍历看看是否能够接上一个字典词构成的字符串
```python
class Solution(object):
    def wordBreak(self, s, wordDict):
        """
        :type s: str
        :type wordDict: List[str]
        :rtype: bool
        """
        if not wordDict or not s:
                return False
        dp = [False for i in range(len(s)+1)]
        dp[0] = True
        
        maxlen = max([len(word) for word in wordDict])
        for i in range(1, len(dp)):
            for j in range(1, min(i, maxlen)+1):
                if dp[i-j] and s[i-j:i] in wordDict:
                    dp[i] = True
        return dp[-1]
```
### 140. Word Break II
#### 返回由字典中单词组成字符串的组合
```
Example 1:
Input:
s = "catsanddog"
wordDict = ["cat", "cats", "and", "sand", "dog"]
Output:
[
  "cats and dog",
  "cat sand dog"
]
```
思路一： 回溯,会超时

```python
class Solution(object):
    def wordBreak(self, s, wordDict):
        """
        :type s: str
        :type wordDict: List[str]
        :rtype: List[str]
        """
        self.res = []
        self.run(s, wordDict, "")
        return self.res
        
    def run(self, s, wordDict, res):
        if len(s) == 0:
            self.res.append(res)
            return
        
        for i in range(len(s)):
            if s[:i+1] in wordDict:
                self.run(s[i+1:], wordDict, res+" "+s[:i+1])
```

```python
class Solution(object):
    def wordBreak(self, s, wordDict):
        """
        :type s: str
        :type wordDict: List[str]
        :rtype: List[str]
        """
        return self.wordBreakHelper(s, wordDict, {})
    
    def wordBreakHelper(self, s, wordDict, memo):
        if s in memo:
            return memo[s]
        if len(s) == 0:
            return ['']
        
        result = []
        
        for word in wordDict:
            if s.startswith(word):
                suffix = s[len(word):]
                for delimitedSuffix in self.wordBreakHelper(suffix, wordDict, memo):
                    if delimitedSuffix == '':
                        result.append(word)
                    else:
                        result.append(word + ' ' + delimitedSuffix)
                        
        memo[s] = result
        return result
```
